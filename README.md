Back-end  
El proyecto utiliza Python con el framework de Django, su base de datos corre usando el ORM de Django y usando el gestor de MySQL.
Se utilizo la API DolarAPI para configurar los endpoint y traer el valor del dolar
Para el web-scraping se usa Beautiful 4 para obtener el valor de la miel por kg

Front-end  
Desarrollado completamente en HTML y CSS.
Integra HTMX para manejar el DOM en la web
Pequeñas porciones de codigo en JavaScript fueron implementadas para manejar modales y asi mejorar la UX/UI
